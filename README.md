# Chef Web App

واجهة ويب عربية حديثة للشيف عبدالرزاق، تتضمن:
- موقعًا عامًا للزوار.
- لوحة إدارة داخل نفس الصفحة.
- تخصيصًا كاملًا للمحتوى مع حفظ البيانات في `localStorage`.

> المشروع مبني كملف ثابت واحد (`index.html`) بدون Build step.

## المزايا الرئيسية
- تصميم UI متكامل ومتجاوب (RTL) مع أقسام: البطل، الخدمات، المعرض، من نحن، التقييمات، والتواصل.
- لوحة إدارة لتعديل النصوص، الصور، الخدمات، وروابط التواصل.
- نظام تسجيل دخول للإدارة مع استرجاع كلمة المرور.
- حفظ جميع الإعدادات والمحتوى محليًا في المتصفح.

## التشغيل المحلي
### الطريقة السريعة
```bash
./run.sh
```
ثم افتح:
- `http://localhost:4173`

### أو مباشرة بدون خادم
يمكن فتح `index.html` مباشرة من المتصفح، لكن تشغيل خادم محلي أفضل للتجربة المتوافقة.

## بيانات الإدارة الافتراضية

## النشر (GitHub Pages)
لضمان التوافق مع إعدادات النشر على GitHub Pages:
1. ارفع الملفات الأساسية كما هي (`index.html`, `README.md`, `run.sh`).
2. من إعدادات المستودع:
   - **Settings → Pages**
   - اختر المصدر المناسب (عادة `Deploy from a branch`).
   - اختر الفرع المطلوب (مثل `main`) والمجلد (`/root`).
3. تأكد أن الصفحة الرئيسية هي `index.html` في جذر المستودع.
